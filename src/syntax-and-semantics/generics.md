# Generics

## Syntax

```
<generic-parameter> ::= <identifier> | "const" <identifier> ":" <type>
<generic-parameters> ::= "[" <generic-parameter> ("," <generic-parameter>)* ","? "]"

<generic-argument>  ::= <type> | <expression>
<generic-arguments> ::= "[" <generic-argument> ("," <generic-argument>)* ","? "]"
```
