# Types

## Syntax

```
<type> ::= <primitive-type>
         | <pointer-type>
         | <reference-type>
         | <slice-type>
         | <array-type>
         | <tuple-type>
         | <struct-type>
         | <variant-type>
         | <named-type>
         | <abstract-type>
```

## Semantics

- Typing is **strong** and **static**.
